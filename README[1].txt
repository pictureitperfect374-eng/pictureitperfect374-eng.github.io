PICTURE IT PERFECT - COMPLETE GITHUB PAGES WEBSITE

Repository:
https://github.com/pictureitperfect374-eng/pictureitperfect374-eng.github.io

FILES TO UPLOAD TO THE MAIN REPOSITORY FOLDER:
- index.html
- styles.css
- script.js
- logo.png

HOW TO INSTALL:
1. Open the repository listed above.
2. Click Add file.
3. Click Upload files.
4. Upload index.html, styles.css, script.js, and logo.png.
5. Replace the older files when GitHub recognizes the matching names.
6. Click Commit changes.
7. Wait one or two minutes.
8. Open the live website and refresh the page.

IMPORTANT:
All four files must remain in the same main/root folder.
